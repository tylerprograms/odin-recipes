# Recipes Project
This is my first HTML project from The Odin Project. The goal was to build a simple recipe website using only HTML and get comfortable with the basics of structuring web pages.

The site includes a main index page that links to individual recipe pages. Each recipe has a short description, a link to a video of someone cooking the recipe, a list of ingredients, and step-by-step instructions.

---

## Skills Demonstrated:
- How to set up a basic HTML structure (boilerplate, headings, paragraphs)
- Linking between multiple pages
- Using ordered and unordered lists
- Organizing content in a clean and readable way

---

## How to View:
1. Clone this repository
2. Open the `index.html` file in your browser
